# BookFeed

Twitterのフィード体験を自炊本の読書に転用するWebアプリ。
複数の本をチャンク分割し、シャッフルして流すことで、SNSを眺める感覚で本が読める。

## 特徴

- **完全ローカル動作** — サーバー不要、`index.html` をブラウザで開くだけ
- **フレームワーク不使用** — HTML/CSS/JS のみ、CDN不要、オフライン対応
- **ダークモード** — 読書に最適な目に優しいデザイン
- **モバイル対応** — スマホファーストのレスポンシブUI

## 使い方

1. `index.html` をブラウザで開く
2. **本棚** → **「本を追加」** からテキストファイル（.txt, UTF-8）をインポート
3. チャンク分割のプレビューを確認して保存
4. **フィード** で複数の本のチャンクがシャッフル表示される
5. 読んだら **「読んだ」** ボタン、あとで読み返したいものは **「ストック」** ボタン

## 画面構成

| 画面 | 機能 |
|------|------|
| フィード | 未読チャンクをシャッフル表示、読了/ストック操作 |
| ストック | お気に入り登録したチャンクの一覧 |
| 本棚 | 登録済みの本の一覧、進捗管理、削除 |

## チャンク分割

- 目標: 2000文字 / チャンク（800〜3500文字の範囲で調整）
- 段落（空行）の境界で分割、文の途中では切らない
- 章区切りに対応（正規表現、空行数、区切り文字列）

## データ保存

- `localStorage` にすべてのデータを保存
- キー: `bookfeed_books`
- サーバー送信なし、データは端末内に留まる

## Google Drive 連携（任意）

Google Drive デスクトップアプリでテキストファイルをローカル同期し、ファイル選択時にその同期フォルダから読み込むことで、複数端末でのファイル管理が可能。

## 技術スタック

- HTML / CSS / JS（Vanilla、フレームワーク不使用）
- localStorage
- FileReader API
